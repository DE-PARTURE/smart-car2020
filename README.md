# SmartCar2020

这是一个智能车项目的阶段性工程集合仓库，包含 OLED 显示移植、小车底盘控制、参数整定、锐角弯处理、赛道化循迹和避障接口测试等多个版本，适合作为智能车开发过程中的阶段记录与代码归档。

## 仓库内容

- `0.96OLED显示屏_STM32F103C8_IIC_V1.0`：OLED 显示屏驱动与移植基础
- `1_舵机转向小车源码V2（库函数版）`：舵机转向小车基础版本
- `2_OLED移植`：OLED 集成后的版本
- `3_示例工程重构`：示例工程重构整理
- `4_参数个性化及直角弯`：参数调优与直角弯处理
- `5_锐角弯`：锐角弯专项版本
- `6_探测循迹结束及循迹数据赛场化`：循迹与赛场化数据处理版本
- `7_避障API测试`：避障相关接口测试版本

## 目录结构说明

多数阶段目录中都能看到类似结构：

- `USER`：应用入口与工程配置
- `CORE`：核心控制逻辑
- `SYSTEM`：系统初始化与基础模块
- `BALANCE`：平衡/控制相关代码
- `HAREWARE` 或 `HARDWARE`：外设驱动
- `STM32F10x_FWLib`：标准外设库
- `修改说明.txt`：该阶段的改动记录

## 开发环境

- MCU：STM32F103 系列
- IDE：Keil MDK
- 固件库：STM32F10x Standard Peripheral Library

## 使用方式

1. 根据你想查看的开发阶段，进入对应目录。
2. 打开其中 `USER` 目录下的 `.uvprojx` 或相关 Keil 工程文件。
3. 结合同目录下的 `修改说明.txt` 理解该阶段的目标与调整内容。
4. 根据自己的车模硬件配置检查引脚和外设初始化后再编译下载。
